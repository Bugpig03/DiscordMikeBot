
# MIKE BOT

Mike Bot is a Discord bot available on the Mike Brant server. It offers several features such as playing local music, telling jokes and "feur" people.




## Version
- 2.5.2 - master
- 2.6.0 - experimental

## Features

- ratio
- jokes
- astrology 
- meteo
- youtube music
- soundboard
- daily message
- web dashboard
